# Financial Deadline Decision Assistant

## Overview

This project is a Python-based decision support system developed to automate deadline calculations for financial payment processing.

The application was created to reduce operational risks caused by manual verification of business rules and dependence on human memory.

## Problem

Financial payment submissions follow specific business rules regarding:

* payment type;
* business days;
* submission deadlines;
* payment availability dates.

An incorrect submission could require the entire process to be restarted.

## Solution

The system:

* calculates business days automatically;
* determines payment availability dates;
* identifies submission deadlines;
* generates real-time alerts;
* provides user-friendly messages.

## Technologies

* Python
* datetime module
* timedelta
* conditional logic
* functions

## Features

* Business-day calculations
* Deadline validation
* Real-time alerts
* User-friendly output
* Operational risk reduction
